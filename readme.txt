说明：
此工具为软件开发格式编写，可以实现ansible批处理效果。
功能强大，支持并发操作、日志生产与日志存储（默认日志级别为DEBUG）。
具体为：（可以在扩展）
1：ssh连接客户端执行命令（类似Xshell）
2：scp功能

Example：
程序名 -u 主机 -g 主机组 -c LINUX命令   #实现ssh远程连接后执行命令
程序名 -u 主机 -g 主机组 -a put/get（上传或下载） -l 本地端路径（绝对路径，包括文件名） -r 远程端路径（绝对路径，包括文件名） #实现scp传输文件


路径说明：
bin目录：执行命令目录 则为Example中的“程序名”
conf目录：配置文件
core目录：核心代码文件
data目录：主机配置文件
lib目录：工具文件（日志）
log目录：存储log（可在配置文件中修改路径）
